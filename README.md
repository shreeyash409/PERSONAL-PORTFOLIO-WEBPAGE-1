The Personal Portfolio Webpage is a responsive and interactive website developed using DHTML, CSS, and JavaScript (Basics). This project showcases my personal information, skills, projects, and contact details in a structured and visually appealing manner.

Technologies Used

DHTML (Dynamic HTML): Used to create dynamic content sections and interactive elements.

CSS: Used for styling, layout design, color schemes, and responsive structure.

JavaScript (Basics): Used to add interactivity such as button actions, navigation handling, and dynamic content updates.

Features

Home section with personal introduction

Skills and technologies overview

Projects section displaying completed tasks

Contact section for user interaction

Responsive and clean UI design

Smooth navigation and basic animations

Purpose of the Project

The main objective of this project is to create a digital identity that represents my skills and work. It helps in learning:

Website structure and layout design

Frontend development fundamentals

DOM manipulation using JavaScript

Creating user-friendly and responsive web pages

Learning Outcomes

Through this project, I gained hands-on experience in:

Designing and developing a complete personal website

Applying HTML, CSS, and JavaScript together

Improving UI/UX design skills

Writing clean and maintainable frontend code
